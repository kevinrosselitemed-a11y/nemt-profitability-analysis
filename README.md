# NEMT Business Profitability Data

This repository contains key financial data from a comprehensive analysis of the Non-Emergency Medical Transportation (NEMT) industry.

## Key Profitability Factors

*   **Reimbursement Rates:** Vary by 55% to 265% depending on state Medicaid policies.
*   **Vehicle Operating Costs:** Typically 30% to 40% of revenue.
*   **Driver Wages & Benefits:** The largest expense at 35% to 45% of revenue.
